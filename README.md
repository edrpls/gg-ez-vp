# GG EZ Video Player

## Usage

```
//CJS
const GgEzVp = require('gg-ez-vp')
const video = new GgEzVp('#div')

// ES
import GgEzVp from 'gg-ez-vp'
const video = new GgEzVp('#div')

// UMD
// index.html
<script src="https://www.some.cdn.com/gg-ez-vp.umd.js">
// index.js
const video = new GgEzVp('#div')
```
